# Gold Economic Impact Analysis

This project analyzes the impact of major economic events (CPI, FOMC, NFP)
on gold price returns using an Event Study methodology.

## Project Structure

## Methodology
- Daily return calculation
- Event window analysis (-5, +5)
- Comparison of returns before, during, and after events

## Tools
- Python
- Pandas
- NumPy
- Matplotlib

## Result
The analysis shows that gold reacts differently to various economic events,
with some effects occurring before the official release date.
